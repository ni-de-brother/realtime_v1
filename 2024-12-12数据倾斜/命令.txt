提交推送
git add .
第二步，查看状态。接下来，我们可以查看文件状态，看看文件是否被添加进了 git 管理：
git status
第三步，提交文件更改。刚才添加的文件只是到了 git 的暂存区，我们还需要提交添加的文件：
git commit -m "input yours message"
第四步，推送。使用 push 命令 将本地仓库推送到远程仓库（Github）中：
git push -u origin master
